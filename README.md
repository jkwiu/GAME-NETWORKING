1. 네트워킹이란?

   2. 지연시간(latency)
      1. 지구 한바퀴를 빛이 도는 시간(133ms = 0.13sec)
         1. 굉장히 빠른 것같지만, 컴퓨터 입장에서는 굉장히 느리다.
      2. 게임에서 얼마나 ``Latency``를 눈속임하느냐가 관건이다.
         1. 아무리 빨라도 물리적인 한계 때문에 속도를 줄이는 것은 안된다.
